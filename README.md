# portfoliothis dashboard, I analyzed sales data and built an interactive dashboard to help users monitor performance and make data-driven decisions.
Dashboard Features:
✅ KPI Cards (Top Sales Region, Top Profit Item, Top Sales Representative) ✅ Sales by Region (Pie Chart) ✅ Profit Trend Analysis (Line Chart) ✅ Sales by Representative (Column Chart) ✅ Top Profit by Product (Bar Chart) ✅ Interactive Slicers for dynamic filtering
Tools Used:
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Key Insights:
📌 Identified the highest-performing sales region. 📌 Tracked profit trends over time. 📌 Compared sales representatives' performance. 📌 Highlighted the most profitable products.
